The hypernested stronger array notation is a notation which allows reducing an expression made up of several numbers into a single extremely large number, for the purpose of googology. Let @ represent an arbitrary (possibly empty) list of non-negative integers which can be separated by any separator of the form (@) or |@|. The comma can be used as a substitute symbol for the separator (0). A valid expression is of the form a{@}, where a can be any non-negative integer and is referred to as the "base". Lastly, let % represent any valid separator (anything of the form (@) or |@|). In order to evaluate an expression into a number, the following rules should be used:

  1. a{} = a + 1
  2. a{@%0} = a{@} (remove trailing zeros)
  3. a{b%@} = a{b - 1%@}{b - 1%@} if b > 0

If none of the rules apply, start this process starting from the first entry inside the curly brackets:

  1. If the entry is equal to 0, jump to the next entry.
  2. If the entry is greater than 0, there must be a 0 preceding it.
    1. If there is a comma between the 0 and the nonzero entry, replace the "0, b" with "a, b - 1" where a is the base and b is the nonzero entry. In simpler terms: a{0,0...,0,0,b,@} = a{0,0...,0,a,b-1,@} if b > 0. The process ends.
    2. If there is a non-comma separator (n%@) where n > 0 between the 0 and the nonzero entry, replace the "0(n%@)b" with "a(n - 1%@)a(n - 1%@)...a(n - 1%@)a(n%@)b-1" with b copies of a before the (n@), where a is the base and b is the nonzero entry. The process ends.
    3. If there is a non-comma separator (0%@) between the 0 and the nonzero entry, simplify the separator itself, removing trailing zeros (analogous to rule 2 of the notation) and/or applying this process on the numbers inside the separator until the separator cannot be simplified anymore. The process ends.
    4. If there is a separator |0| between the 0 and the nonzero entry, replace the "0|0|b" with "...a(a(a)a)a..." with b nesting levels, where a is the base and b is the nonzero entry. The process ends. (To clarify what "b nesting levels" means: 1 nesting level is a, 2 nesting levels is a(a)a, 3 nesting levels is a(a(a)a)a, etc.)
    5. If there is a separator |n%@| where n > 0 between the 0 and the nonzero entry, replace the "0|n%@|b" with "a|n - 1%@|a|n - 1%@|...a|n - 1%@|a|n - 1%@|b-1" with b copies of a before the (n@), where a is the base and b is the nonzero entry. The process ends.
    6. If there is a separator |0%@| between the 0 and the nonzero entry, simplify the separator itself, removing trailing zeros (analogous to rule 2 of the notation) and/or applying this process on the numbers inside the separator until the separator cannot be simplified anymore. The process ends.

The application of the rules and/or process should be repeated until the expression is reduced to a number.
