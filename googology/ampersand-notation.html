Ampersand Notation

Ampersand Notation (&N) is a notation that can express extremely large numbers. The notation is currently composed of 3 parts, which are as follows:

  • Basic Ampersand Notation (B&N)
  • Extended Ampersand Notation (X&N)
  • Nested Ampersand Notation (N&N)

I plan to create the following parts in the future:

  • Hyper-Nested Ampersand Notation (HN&N)

Number names are available here.
