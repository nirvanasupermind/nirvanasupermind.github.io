Stacked Quetta- Prefix System (SQPS)

This is my proposed extension of the SI prefix system for prefixes beyond quetta- (1030). Richard J.C. Brown (the proposer of the prefixes ronna- and quetta-) suggested that beyond 1030, compound prefixes with the last prefix as "quetta-" should be used, for example "kiloquetta-" for 1033, although this is not approved by the BIPM due to a lack of need for these prefixes. However, this system results in cumbersome names starting at 1060, like "quettaquetta-" for 1060, "quettaquettaquetta-" for 1090, and so on. I have employed Greek prefixes to make these shorter, such as "diploquetta-" (double quetta-) for 1060 and "triploquetta-" (triple quetta-) for 1090. Large SI prefixes themselves are based on Greek prefixes, which means this system can nest into itself like "quettaploquetta-" (1030 times quetta-) for 103×1031.

Additionally, this system adds prefixes for powers of 10 that were skipped in the base system (such as 104 and 105) which is achieved by using compound prefixes beginning with "deca-" and "hecto-" or "deci-" and "centi-".
