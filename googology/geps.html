Greek Extended Prefix System (GEPS)

This is my proposed extension of the SI prefix system for prefixes beyond quetta- (1030).

Number Latin root Symbol
1 hena h
2 di d
3 tri r
4 tetra t
5 penta p
6 hexa x
7 hepta h
8 octa o
9 ennea e
10 deca de (d if there is a nonzero ones place)
20 icosi i
30 triaconta rc (r if there is a nonzero ones place))
40 tetraconta tc (t if there is a nonzero ones place))
50 pentaconta pc (p if there is a nonzero ones place))
60 hexaconta xc (x if there is a nonzero ones place))
70 heptaconta hc (h if there is a nonzero ones place))
80 octaconta oc (o if there is a nonzero ones place))
90 enneaconta ec (e if there is a nonzero ones place))
100 hecto he

For powers of 1,000 starting at 1,000, the "Greek root" is just the SI/EGPS prefix for that power of 10 and the symbol being its SI prefix symbol. For multiples, you stack roots-for example 200 is "dihecto", 30,000 is "triacontakilo", 200,000 is "dihectokilo" etc.
