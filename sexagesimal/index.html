We Want To Replace Decimal With Decimal-Coded Sexagesimal

Similar to how our decimal system is based on the powers of 10, the sexagesimal numeral system is based on powers of 60.

Pure sexagesimal requirees 60 single digits for the numbers 0-59, much like how decimal requires 10 digits for 0-9. 60 digits is a very large amount, and makes memorizing the multiplications of all single-digit numbers next to impossible. However, we can avert this by using a system known as "decimal-coded sexagesimal", where each sexagesimal digit is represented as a pair of decimal digits, ranging from 00-59.
